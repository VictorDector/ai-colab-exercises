# AI Colab Exercises

Repositorio de ejercicios y prácticas de Inteligencia Artificial en Google Colab (notebooks .ipynb).
Objetivo: documentar y reproducir los pasos y resultados de los ejercicios del máster/curso.

## Estructura
- `00_setup/` — plantillas, utilidades, funciones comunes
- `01_supervised/` — aprendizaje supervisado (árboles, regresión, SVM, etc.)
- `02_unsupervised/` — aprendizaje no supervisado (clustering, reducción de dimensión, etc.)
- `03_deep_learning/` — redes neuronales y deep learning
- `04_nlp/` — procesamiento de lenguaje natural
- `case_studies/` — casos prácticos completos

## Cómo ejecutar
### Opción A: Abrir en Google Colab
1. Abrir el notebook `.ipynb` desde GitHub.
2. Copiar la URL del notebook.
3. Ir a https://colab.research.google.com y usar **File → Open notebook → GitHub**.
4. Pegar la URL o buscar el repositorio.

### Opción B: Ejecutar local (opcional)
Requiere Python 3.10+ y Jupyter.
1. `pip install -r requirements.txt` (si existe)
2. `jupyter notebook`
